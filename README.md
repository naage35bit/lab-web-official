# lab-web-official

本仓库用于归档和发布多个独立的 HTML 页面。这些页面是独立开发的 Web 资源，不隶属于任何特定域名或单一网站项目。

## 项目简介

`lab-web-official` 是一个静态 HTML 页面集合仓库。这里的每个页面都可以独立运行，适用于展示、测试或作为 Web 技术实验的记录。仓库本身不绑定任何特定用途，仅作为页面归档与发布的统一入口。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/`：主要存放各个独立的 HTML 页面文件。
- `assets/`：存放页面间共享的样式、脚本或媒体文件，可按需组织子目录。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立开发，彼此之间无强依赖关系。页面内容、样式和功能由开发者自行定义，仓库仅负责托管与版本管理。

如需添加新页面，请遵循以下建议：

1. 在 `pages/` 目录下新建 HTML 文件。
2. 确保页面引用资源路径正确（建议使用相对路径）。
3. 如有公共资源，请放置于 `assets/` 目录下并统一引用。
